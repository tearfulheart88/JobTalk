# 진로톡 심사위원 30초 가이드

## 한 문장

진로톡은 취업 정보만 나열하지 않고, 청년의 시간·비용·경험·불안 장벽을 **오늘 할 수 있는 작은 행동**으로 바꾸는 커리어 실행 MCP입니다.

## 첫 질문

> 백엔드 개발자로 취업하고 싶은데 알바를 병행해서 하루 20분밖에 없고 경력도 없어요. 어디서부터 시작할까요?

진로톡은 `build_career_action_plan`으로 다음을 한 번에 제공합니다.

1. 오늘 20분 미션과 명확한 완료 기준
2. 막혔을 때 10분 축소 행동
3. 공고 비교와 자기소개서까지 이어지는 7일 계획
4. 채용공고·청년정책·직무분석·첨삭으로 연결되는 다음 질문

## 평가 기준 대응

| 기준 | 진로톡의 답 |
|---|---|
| 창의성 | 정보 검색을 넘어 취업 장벽을 실행 가능한 미션으로 변환 |
| 편의성 | 한 번의 자연어 요청으로 오늘 행동과 7일 여정 제공 |
| 안정성 | API 키 없이도 대표 기능 작동, 데모/실데이터 구분, 호출량·비밀값 보호 |

## 구현 경계

- 실제 구현: 7일 실행계획, 사람인 채용 검색, 온통청년 정책 검색, 직무 분석, 자소서·면접 코칭
- API 키가 없을 때: 검색·LLM 결과는 `데모` 또는 `mock`으로 명확히 표시
- 현재 미구현: 멘토 연결, 지원서 자동 제출, 채용 알림 자동 발송
- 개인정보: 실행계획에는 이름·전화번호·주민번호가 필요하지 않으며 외부 API를 호출하지 않음
