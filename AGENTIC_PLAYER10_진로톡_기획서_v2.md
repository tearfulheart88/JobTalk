## 1. 아이디어 선정 근거

### 1.1 돌봄톡과의 차별점

| 구분 | 돌봄톡 (CareTalk) | 진로톡 (CareerTalk) |
|------|-------------------|---------------------|
| **타겟** | 독거노인 (65세 이상, 약 170만명) | 청년·학생 (15~29세, 약 797만명) |
| **도메인** | 고령자 돌봄·안부확인·생활지원 | 진로·취업·멘토링·자기계발 |
| **페인포인트** | 사회적 고립, 건강 모니터링 부재 | 취업 불안, 진로 정보 부족, 멘토 접근성 낮음 |
| **톤앤매너** | 따뜻하고 느린 말투, 큰 글씨 | 친근하고 활기찬 말투, 이모지 활용 |
| **데이터 소스** | 복지부 API, 지역 복지관 DB | 사람인 OpenAPI, 온통청년 API, 청년정책 DB |
| **Widget UX** | 큰 버튼·음성 입력·긴급 호출 | 카드형 공고·진로 로드맵·멘토 매칭 |

→ **완전히 다른 타겟·도메인으로 심사위원에게 다양성 어필**

**카카오톡 침투율**

- 청년층은 카카오톡을 일상 커뮤니케이션의 기본 채널로 사용
- → 카카오톡 기반 진로 서비스 = **접근성 장벽 제로**

### 1.3 1회 수상작 분석 — SeedUp과의 차별화

| 구분 | SeedUp (1회 2등) | 진로톡 (CareerTalk) |
|------|------------------|---------------------|
| **타겟** | 창업가·스타트업 | 청년 구직자·학생 (더 넓은 층) |
| **기능** | 정부 지원사업 탐색 | 채용공고 + 진로멘토링 + 자기소개서 코칭 |
| **데이터** | 지원사업 공고 | 채용공고(사람인) + 청년정책(온통청년) + 멘토DB |
| **차별점** | — | 진로 탐색(What) + 취업 실행(How) 통합 |

→ SeedUp이 "창업가의 지원사업 탐색"이라면, 진로톡은 **"청년의 진로 탐색부터 취업 실행까지"** 풀스택

### 1.4 "왜 카카오톡에서 이걸 해야 하는가" — 6가지 논리

1. **접근성**: 청년 95%+가 매일 사용하는 플랫폼 → 별도 앱 설치 장벽 제거
2. **불안 해소**: 32.4%의 '막막한 불안감'을 AI와의 익명 대화로 경감 (카톡의 친숙한 UI)
3. **정보 비대칭 해소**: 경쟁사 모두 정보 과잉 또는 부실 → AI 큐레이션으로 차별화
4. **진로 선택 지원**: 31.7%가 진로 결정에 어려움 → 카톡 대화형 AI 진로 컨설팅
5. **틈새 공고 발굴**: 워크넷 부실·원티드 편중 → 사람인 OpenAPI로 중소·지역 공고까지 커버
6. **지속적 관계**: 취업포털은 '방문 후 이탈' → 카톡은 일상적 관계 유지 → 멘토링·알림·후속 관리 가능

### 2.2 MCP Tool — MVP 4종 (현실적 범위)

| # | Tool 이름 | 설명 | 입력 | 출력 | 데이터 소스 |
|---|----------|------|------|------|------------|
| T-01 | `search_jobs` | 맞춤 채용공고 검색 | 직무, 지역, 경력, 키워드 | 공고 리스트 (회사·직무·마감일·링크) | 사람인 OpenAPI (JSON, 500회/일) |
| T-02 | `analyze_job_fit` | AI 진로 적성 진단 및 직무 추천 | 관심분야, 학력, 성향 키워드 | 추천 직무 TOP5 + 진입 경로 | Claude/GPT API (Prompt Caching + Streaming) |
| T-03 | `search_youth_policies` | 청년정책·지원금 매칭 | 나이, 지역, 상황(재학/졸업/미취업) | 매칭 정책 리스트 + 신청 링크 | 온통청년 OpenAPI (실시간, 무료, JSON) |
| T-04 | `generate_resume_tip` | 자기소개서 첨삭 + 면접 코칭 | 자소서 텍스트, 지원 직무 | 첨삭본 + 예상 면접질문 5선 | Claude/GPT API (경량 모델) |

> **MVP 범위 조정 근거**: 4주 내 5종 구현은 과도. Tool당 3~5일 소요. 고용24 API는 기업회원 전용으로 사용 불가 → 사람인+온통청년으로 대체 완료. `mentor_connect`는 시간 남을 경우에만 추가.

### 2.3 LLM 응답 지연 해결 방안 (MVP 적용)

| 방안 | 지연 감소 | 비용 절감 | 구현 난이도 | MVP 적용 |
|------|----------|----------|------------|----------|
| **Prompt Caching** | 최대 80~85% | 30~90% | 중간 | ⭐⭐⭐⭐⭐ 최우선 |
| **Streaming 응답** | 체감 50~70% | 동일 | 낮음 | ⭐⭐⭐⭐⭐ 필수 |
| **경량 모델 라우팅** | 40~60% | 50~90% | 낮음 | ⭐⭐⭐⭐ |
| **프롬프트 최적화** | 20~40% | 비례적 | 중간 | ⭐⭐⭐⭐ |

**예상 개선 효과**

| 시나리오 | 적용 전 | 적용 후 | 개선율 |
|----------|---------|---------|--------|
| 채용공고 분석 | 8~12초 | 2~4초 | 60~75% |
| 자소서 피드백 | 5~8초 | 1.5~3초 | 50~65% |
| 진로 추천 | 6~10초 | 2~4초 | 55~70% |

**구현 전략**
- Claude: 시스템 프롬프트 + 반복 컨텍스트 캐시 마킹 → 최대 90% 입력 비용 절감, TTFT 20~80% 개선
- GPT: 1024토큰 이상 접두사 자동 캐싱 → 50% 할인
- FastMCP에서 `stream=True` → 토큰 단위 실시간 출력 → TTFT 0.5~1초대 체감
- 단순 쿼리: GPT-4o-mini / Claude Haiku, 복잡 분석: GPT-4o / Claude Sonnet

## 3. Widget 기획 (청년용 UI) — 카카오톡 생태계 통합

### 3.1 Kakao Tools Widget 제약사항 분석

| 제약 | 상세 | 진로톡 대응 |
|------|------|------------|
| **렌더링 한계** | 인라인 카드/리스트만 가능. 지도 인터랙션(확대/축소)은 KakaoMap 앱 전환 필요 | 카드형 UI로 충분히 커버 가능 |
| **위젯 타입 제한** | 현재 4종만 (KakaoMap, Reservations, Gift, Melon). **커스텀 위젯 추가 불가** | 카카오가 직접 통제 → 우회 전략: 카카오톡 채널 + 알림톡/친구톡 + 카카오 로그인 조합 |
| **인터랙티브 UI** | 버튼·폼·모달 미지원. 정적 카드 + 딥링크만 | 카드에 딥링크 버튼 포함 → 외부 앱/웹 전환 |
| **응답 시간** | ChatGPT 추론 + Kakao Tool API 호출 시간 추가 | Prompt Caching + Streaming으로 상쇄 |
| **제3자 Open API** | **현재 제공되지 않음**. 외부 개발자 위젯 등록 불가 | 독자 채널 기반 서비스로 차별화 |
| **확장 계획** | 카카오 "향후 외부 파트너·공공기관 협력 확장 예정" 발표했으나 2026.06 기준 추가 위젯 없음 | 향후 가능성에 대비한 설계 |

### 3.2 우회 전략 — 카카오톡 채널 + API 조합

```
[사용자] → 카카오톡 채널 '진로톡' 친구 추가
         → 카카오싱크 간편가입
         → 진로톡 MCP 서버 (PlayMCP)
              ├── 카카오 로그인 API (인증)
              ├── 채널 관계 조회 API (구독 상태)
              ├── BizMessage 알림톡 API (맞춤 진로 정보)
              ├── 카카오맵 로컬 API (진로센터 검색)
              ├── 카카오 내비 API (면접 길찾기)
              └── 카카오페이 API (유료 상담 결제)
```

### 5.2 진로톡의 경쟁 우위

1. **카카오톡 네이티브**: 앱 설치 불필요, 청년층 97.8% 사용 중인 플랫폼
2. **진로탐색→취업실행 풀스택**: 적성진단 → 공고추천 → 자소서코칭 → 면접준비 원스톱
3. **청년정책 연동**: 온통청년 API로 수당·지원금·교육 프로그램까지 통합 (타 서비스 미제공)
4. **AI 멘토링**: 24/7 즉시 응답, 익명성 보장으로 진입장벽 낮춤
5. **중소·지역 공고 발굴**: 워크넷 부실·원티드 편중의 틈새 — 사람인 OpenAPI로 양질의 중소·중견·지역기업 공고 큐레이션
6. **지속적 관계**: 취업포털은 '방문 후 이탈' → 카톡은 일상적 관계 유지 → 멘토링·알림·후속 관리 가능

## 6. 심사 전략

### 6.1 돌봄톡과의 시너지 전략

| 전략 | 내용 |
|------|------|
| **타겟 다양성** | 돌봄톡(시니어) + 진로톡(청년) = 전 세대 커버 → "카카오 생태계 확장성" 어필 |
| **기술 스택 재사용** | 동일 MCP 프레임워크·Widget 패턴 → "개발 생산성·확장성" 증명 |
| **도메인 상호보완** | 돌봄(사회안전망) + 진로(경제활동) = "공공성 + 실용성" 양축 |

### 6.2 심사 기준별 공략 포인트

| 심사 기준 | 진로톡 공략 |
|-----------|------------|
| **창의성** | "카카오톡에서 AI 진로 멘토를 만난다" — 기존 채용플랫폼과 완전히 다른 접근. Kakao Tools Widget의 폐쇄성을 역이용한 독자 채널 전략 |
| **편의성** | 앱 설치 없이 카톡 대화만으로 진로탐색·취업준비 완결. 카카오싱크 간편가입으로 온보딩 10초 |
| **안정성** | 사람인 공식 API + 온통청년 공공 API 기반 → 데이터 신뢰성 확보. Prompt Caching + Streaming으로 LLM 응답 2~4초대 |
| **파급력** | 청년 797만명 + 대학생 200만명 = 잠재 사용자 500만명 규모. TAM 797만명, SAM 142만명, SOM 5~12만 MAU (3년 목표) |

### 6.3 2개 동시 제출 전략

- 돌봄톡: **공공성·사회적 가치** 강조 (독거노인 문제 해결)
- 진로톡: **실용성·대중성** 강조 (청년 취업 문제 해결)
- → 심사위원에게 **"서로 다른 가치를 가진 두 서비스"**로 인식시켜 수상 확률 극대화

### 6.4 카카오 생태계 통합 어필 포인트

- **카카오 로그인·싱크**: 간편가입 + 채널 추가 동시 처리 → 온보딩 마찰 제로
- **BizMessage 알림톡**: 맞춤 공고·마감 임박·면접 일정 알림 → 리텐션
- **카카오맵·내비**: 진로센터·면접장소 검색 및 길찾기 → 오프라인 연계
- **카카오페이**: 유료 진로상담·멘토링 결제 → 수익 모델
- **카카오워크**: 조직도 연동 → 기업 내 멘토링 프로그램 확장

## 9. 타겟 페르소나 (User Journey 기반)

### 페르소나 A: "막막한 취준생" (20~24세, 대학 재학·졸업예정)

| 여정 단계 | 행동 | 감정·페인포인트 |
|-----------|------|----------------|
| ① 진로 고민 시작 | "뭘 해야 할지 모르겠다" → 포털 검색 | **혼란**, 정보 과잉 |
| ② 취업포털 방문 | 잡코리아·사람인 검색 → 수천 개 공고 | **압도됨**, 필터링 불가 |
| ③ 무작위 지원 | "일단 넣어보자" 식 지원 → 서류 탈락 | **불안**, 자존감 하락 |
| ④ 막막함 심화 | "취업준비 방법을 모르겠다" (39.2%) | **막막함**, 방법론 부재 |

**핵심 니즈**: 진로 방향성 제시, 맞춤형 공고 추천, 취업 준비 로드맵

### 페르소나 B: "지친 장기 취준생" (25~29세, 졸업 후 1년+ 구직)

| 여정 단계 | 행동 | 감정·페인포인트 |
|-----------|------|----------------|
| ① 졸업 후 구직 | 본격 취업활동 → 수십 회 지원 | **의욕**, 기대 |
| ② 반복된 실패 | 서류 탈락 반복 → "스펙이 부족한가" | **불안**(32.4%), 자책 |
| ③ '쉬었음' 상태 | 구직 포기 → 50.4만명 중 하나 | **무기력**, 사회적 고립 |
| ④ 재도전 or 포기 | 눈높이 낮추기(53.5%) or 구직단념(39만명) | **체념**, 구조적 니트 위험 |

**핵심 니즈**: 심리적 지원(불안 해소), 실질적 합격 전략, 소규모·틈새 기회 발굴

### 페르소나 C: "틈새 탐색자" (22~27세, 전문대·지방대 졸업, 중소기업 취업 후 이직 희망)

| 여정 단계 | 행동 | 감정·페인포인트 |
|-----------|------|----------------|
| ① 첫 취업 | 중소기업 취업 → 낮은 연봉·복지 | **불만족**, "더 나은 곳 있을 것" |
| ② 이직 탐색 | 잡코리아·사람인 검색 → 대기업 공고만 눈에 띔 | **정보 비대칭**, 소외감 |
| ③ 숨은 공고 발굴 실패 | 지역 중소·중견 공고 찾기 어려움 | **좌절**, "워크넷은 부실공고" |
| ④ 정체 | 현 직장 유지 or 무작위 이직 | **정체감**, 커리어 경로 불투명 |

**핵심 니즈**: 중소·중견·지역기업 양질의 공고 큐레이션, 연봉·기업리뷰 투명성, 커리어 경로 설계

## 10. 시장 규모 추정 (TAM/SAM/SOM)

### TAM (Total Addressable Market)
- 15~29세 청년 인구: **797.4만명** (2025.05 국가데이터처)
- 15~39세 확장 청년: 약 1,300만명 (청년기본법 기준)
- 보수적 TAM: **797만명**

### SAM (Serviceable Addressable Market)
- 경제활동참가율 49.5% → 경제활동 청년: **395만명**
- 실업자 + 취업준비자 + 구직단념자 + '쉬었음' 중 구직의사자: 약 **150만명**
- KakaoTalk 사용률 95%+ → SAM: **142만명** (구직 관련 청년 중 카톡 사용자)
- 보수적 SAM: **120~150만명**

### SOM (Serviceable Obtainable Market, 3년 목표)
- KakaoTalk 채널 기반 신규 서비스 침투율: 3~5% (초기 3년)
- 142만 × 3% = **4.3만 MAU** (보수)
- 142만 × 5% = **7.1만 MAU** (중간)
- 395만 × 3% = **11.9만 MAU** (낙관, 전체 경제활동 청년 대상)
- **SOM 3년 목표**: **5~12만 MAU**

### 시장 규모 참고
- 사람인 2021년 매출 1,290억원, 영업이익 391억원
- 잡코리아+알바몬 합산 점유율 53.1%
- 온라인 채용 시장 전체: 연 3,000억원+ 추정
- 진로톡은 채용 중개 수수료가 아닌 **AI 진로 컨설팅·멘토링** 모델 → 인당 ARPU 1~3만원/년 가정 시 SOM 매출 5~36억원/년
